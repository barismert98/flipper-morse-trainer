# Changelog

## 1.0

Initial release.

- Encoding / Decoding / Mixed modes with independent per-mode progress
- 13-level curriculum with challenge word levels and rotating vocabulary pools
- Free Practice mode with live decoding and playback
- Adaptive review weighting based on per-letter miss history
- Hold-to-key input with sidetone, real-time LED feedback, and dot/dash threshold bar
- Farnsworth-spaced playback option, speed ramp on later levels
- Hints (50/50 and letter reference card), stars, daily streak, lifetime stats
- Settings: sound, vibration, letter gaps, visuals, LED, reset progress
